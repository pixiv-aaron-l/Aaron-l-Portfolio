\# Aaron-l Portfolio



A personal portfolio website and local administration application.



\## Features



\- Portfolio management

\- Album management

\- Artwork management

\- Blog system

\- Static website generation

\- Attachment downloads

\- Automatic cleanup of obsolete generated files



\## Technologies



\- Python

\- PySide6

\- HTML

\- CSS

\- JSON



\## Structure



```

admin/

website/

```



The admin application edits JSON data and generates a complete static website.



The generated website can be deployed to static hosting.
